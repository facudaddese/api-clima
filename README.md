
# Weather App

Proyecto que demuestra el consumo de APIs REST mediante JavaScript vanilla. Desarrollado con **Fetch API** y manejo asíncrono para obtener información meteorológica de **OpenWeatherMap**. [Ver sitio](https://facudaddese.github.io/api-clima/)

---

## Funcionalidades

- Búsqueda de clima por **ciudad o país**
- Temperatura actual en grados Celsius
- Velocidad del viento (km/h)
- Ráfagas de viento (si están disponibles)
- Sensación térmica
- Validación de campo vacío
- Notificaciones con **Toastify**
- Mensaje de error cuando la ciudad o país no existe

---

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API & Async/Await
- OpenWeatherMap API
- Toastify.js

---

## API utilizada

Se utiliza la API pública de **OpenWeatherMap** para obtener los datos climáticos.

**Documentación oficial:**  
[OpenWeather Current Weather API](https://openweathermap.org/current)
